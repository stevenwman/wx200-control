# wx200-control

Using [`mink`](https://github.com/kevinzakka/mink/tree/main) for inverse kinematics control in [MuJoCo](https://github.com/google-deepmind/mujoco).

WX200 CAD from [Trossen Robotics](https://docs.trossenrobotics.com/interbotix_xsarms_docs/specifications/wx200.html), converted to SLDASM in SolidWorks, then uploaded to OnShape, assmembled, and exported as MJCF using [`Onshape-to-robot`](https://onshape-to-robot.readthedocs.io/en/latest/).

WS200 Robot hardware interface using [`dynamixel-sdk`](https://pypi.org/project/dynamixel-sdk/).

SpaceMouse support using [PySpaceMouse](https://github.com/JakubAndrysek/PySpaceMouse).

Check `requirements.txt` for dependencies
